‚öôÔ∏è How The System Calculates

Formula:

Program Day Number = Days Since Start Date
Day of Cycle = Program Day Number % 7
Activities = Cycle Day Template (0=Monday, 1=Tuesday, etc.)

Example Calculation for Jan 8 (if you started Jan 6):

  1. Days since start: Jan 8 - Jan 6 = 2 days
  2. Day of cycle: 2 % 7 = 2
  3. Cycle day 2 = Wednesday's template
  4. Display: Desk Resets + Evening Gym
